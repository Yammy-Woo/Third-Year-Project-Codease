This is my third-year project,  Codease, a web-based application that helps users with little or no coding experience create and customise their own websites through an easy-to-use interface. The platform includes a Template Editor that allows users to edit elements such as navigation bars, text, and media, as well as access a library of pre-designed templates for quick setup. The system focuses on accessibility by using clear labels and simple icons to improve user experience. Built using React.js and Django, with JavaScript used to implement interactive features such as drag-and-drop and dynamic resizing of elements.
